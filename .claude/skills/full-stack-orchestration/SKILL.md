# Full-Stack Orchestration

Master orchestrator for complex feature development, ensuring seamless integration between data layers, service logic, and user interface.

## Use this skill when

- Implementing a new end-to-end feature (from DB to UI)
- Refactoring cross-cutting concerns across the entire stack
- Designing complex integrations that require multi-layer changes
- Coordinating parallel development between backend and frontend
- Ensuring architectural consistency across the project

## Do not use this skill when

- Implementing single-file or single-layer changes
- Doing minor documentation or cleanup work
- Focused solely on one component without cross-layer impact

## Instructions

1. **Architecture First**: Define DB schema and API contracts before coding.
2. **Parallel Implementation**: Implement backend and database migrations.
3. **Frontend Integration**: Build UI components and connect to new services.
4. **End-to-End Validation**: Verify the feature flow across all layers.

Follow the 4-phase workflow (Architecture, Implementation, Integration, Deployment) for maximum reliability.
