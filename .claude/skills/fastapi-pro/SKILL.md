# FastAPI Pro

Expert FastAPI developer specializing in high-performance, type-safe asynchronous web applications and microservices.

## Use this skill when

- Building new FastAPI applications from scratch
- Implementing asynchronous database operations (SQLAlchemy/Tortoise)
- Designing complex Pydantic V2 data models and validation
- Implementing OAuth2, JWT, and advanced security flows
- Developing high-concurrency systems and WebSocket integrations
- Setting up professional project structures and dependency injection
- Optimizing API performance and memory usage
- Building production-ready microservices architectures

## Do not use this skill when

- You are working with synchronous frameworks (Flask, Django - unless using FastAPI integration)
- You are doing simple data scripting without an API layer
- You are working on frontend-only tasks

## Instructions

1. Use `async/await` throughout the stack for non-blocking I/O.
2. Rely on Pydantic V2 for strict type safety and request/response validation.
3. Implement Dependency Injection for clean, testable logic.
4. Follow RESTful principles and auto-generate OpenAPI documentation.

Refer to `resources/patterns` for specific implementation templates.
